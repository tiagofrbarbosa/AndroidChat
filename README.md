# AndroidChat
Chat application for android


## To start
The api used in this project is available in api / ichat-api.jar, to run use java -jar ichat-api.jar,
In chatModule.jar insert your ip


## Used libraries
- ButterKnife
- Dagger
- EventBus
- Retrofit
- Picasso
